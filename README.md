# 마크다운( markdown ) 사용법(문법)  
마크다운을 좀 더 잘 활용해보고 싶은데, 매번 검색하기 귀찮아서 내가 자주 쓸만한 것을 정리해놓은 것이다.  
## 1. 헤더Headers  
h1부터 h6까지 **'#'** 을 붙여 사용 가능하다.  

    # This is a h1.
    ## This is a h2.  
    ### This is a h3.  
    #### This is a h4.  
    ##### This is a h5.  
    ###### This is a h6.  

적용 예 :   
*** 
# This is a h1.
## This is a h2.  
### This is a h3.  
#### This is a h4.  
##### This is a h5.  
###### This is a h6.  
***
